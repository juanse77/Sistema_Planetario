<h1>Sistema solar:</h1>

<p>En este proyecto se ha buscado generar una simulación del sistema solar. Los valores de distancias orbitales y el tamaño de los planetas son simplificaciones para que la animación sea más visual. No se ha pretendido hacer un sistema solar dimensionalmente realista.</p>

<p>La animación permite mover la cámara para que enfoque desde la vista superior, a la frontal y a la inferior. Las teclas para mover la cámara son:</p>

<ul>
	<li>La tecla 'subir' para subir el ángulo de la cámara.</li>
	<li>La tecla 'bajar' para bajar el ángulo de la cámara.</li>
</ul>

<p>También se permite pausar la animación. Para ello se utiliza la tecla 'espacio'</p>

<p>La aplicación se ha realizado en Processing y en ella se ha hecho un uso intensivo de los métodos pushMatrix, popMatrix, rotate y translate.</p>
